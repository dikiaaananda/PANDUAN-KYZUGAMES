PANDUAN CARA MASUKIN GAME:

PANDUAN VIDEO LENGKAP

1. Download File

Download data game dan alat sinkronisasi yang dikirim dari telegram.

2. Install Alat Sinkronisasi

Install alat sinkronisasi dan buka alatnya (akan muncul ikon Steam mengambang).

3. Ekstrak Data Game

Ekstrak data game dan buka filenya. Ada dua tipe (tanpa DLC dan dengan DLC). Jika di data game ada folder DLC, buka yang folder database.

4. Select All File

Select all (pilih semua) file yang ada di dalam folder tersebut.

5. Drag ke Ikon Steam

Drag semua file ke ikon Steam yang tadi (akan muncul notif berhasil).

6. Restart Alat

Klik kanan ikon Steam mengambang tadi kemudian restart.

6. GAME SUDAH MASUK DI AKUN ANDA
